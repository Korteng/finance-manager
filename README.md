Finance Manager

Текущее состояние
Базовая инфраструктура v1.0.1
(Технический релиз без функционала)

Что реализовано:
- Настроен CI/CD (Maven + GitHub Actions)
- Добавлены Git-ветки: `main`, `dev`, `feature/*`
- Базовый `pom.xml` с Spring Boot 3.x
- Автотесты (шаблон)

Ограничения:
- Нет рабочего кода
- Отсутствуют API-эндпоинты
- Не подключена БД

Инструкция
bash
Сборка
mvn clean package

Запуск
java -jar target/*.jar
